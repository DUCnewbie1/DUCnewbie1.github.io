Install Ansible

You will now install Ansible on your EC2 “AnsibleServer” instance.

  1. First, connect to EC2 AnsibleSever similar to the step Install Jenkins
  • Select Project: Maven –> Language: Java –> SpringBoot: 3.3.3 –> Packaging: Jar –> Java: 22
  • Execute the command:
sudo amazon-linux-extras install ansible2
  • Check the ansible version, we use the command:
ansible --version
  1. Configure Ansible: Kubernetes Server, SSH key
  • Go to Dashboard → Jenkins Management → System Configuration → Available Plugins
  • Then enter Publish via SSH in the search box.

  • Then click Install to install.

  • Go to Dashboard → Jenkins Management → System Configuration → Publish via SSH. Enter all the necessary data of the EC2 Ansible Server in this section:

  • After clicking the Apply and Save buttons

  • Next click on the configuration test section to confirm that the Plugin is working.
